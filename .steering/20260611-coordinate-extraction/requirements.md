# 要求内容

## 概要

MVP 0として、コーディネート画像から画像生成向けのコーデ抽出プロンプトを作成し、ChatGPTで同じ画像から抽出した結果と比較評価できるようにする。

この作業では、画像生成APIの呼び出しや最終画像生成プロンプトの合成には進まず、コーデ抽出品質の検証に集中する。

## 関連 Issue

https://github.com/kanan4gh/platform-harness-for-codex/issues/2

## ユースケースの軸

ユーザーとして、コーディネート画像を入力して服装・小物・色・素材・シルエットを画像生成向けプロンプトとして抽出し、ChatGPTの抽出結果と比較したい。

## 実装対象の機能

### 1. コーディネート画像入力

- 実写画像またはアニメ風イラスト画像をコーデ抽出対象として指定できる
- 入力画像のパス、入力種別、任意のメモを抽出セッションとして扱える
- 存在しないファイル、対応外形式、読み込み不能な画像に対して分かりやすいエラーを返す

### 2. コーディネート抽出プロンプト生成

- 入力画像から、服装・小物・色・柄・素材感・シルエット・季節感・スタイリングを抽出するためのCodex向け指示を実行できる
- 抽出結果を、画像生成画面へ貼り付けやすいコーデ再現プロンプトとして保存・表示できる
- アニメ風イラスト画像の場合は、画風ではなく服装構成を実写生成にも転用しやすい形で抽出する
- 不確実な要素は断定せず、曖昧さが分かる表現を維持する

### 3. ChatGPT比較評価記録

- ユーザーがChatGPT Web UIで得たコーデ抽出結果を貼り付けて保存できる
- 本ツールの抽出結果とChatGPTの抽出結果を、同じ抽出セッションに紐づけて比較できる
- 比較観点として、服装要素の網羅性、色・柄の正確さ、素材感の表現、シルエットの表現、画像生成プロンプトとしての使いやすさを記録できる
- 比較結果として、同等・不足・過剰・要改善の評価と自由記述メモを保存できる

### 4. 抽出セッションのローカル保存

- 入力画像情報、本ツールの抽出結果、ChatGPT比較結果、評価メモをローカルファイルとして保存できる
- 保存済みセッションを後から参照できる
- MVP 0ではJSONまたはYAMLファイルによる保存を優先し、SQLiteは導入しない

## 受け入れ条件

### コーディネート画像入力

- [ ] ユーザーは実写画像ファイルを抽出対象として指定できる
- [ ] ユーザーはアニメ風イラスト画像ファイルを抽出対象として指定できる
- [ ] 入力種別を `photo` または `illustration` として指定できる
- [ ] 存在しない画像パスを指定した場合、抽出処理に進まずエラーを表示する
- [ ] 対応外拡張子を指定した場合、抽出処理に進まずエラーを表示する

### コーディネート抽出プロンプト生成

- [ ] 画像からトップス、ボトムス、靴、バッグ、アクセサリーなどの主要要素を抽出する指示を実行できる
- [ ] 色、柄、素材感、シルエット、季節感、スタイリングの特徴を含む抽出結果を保存できる
- [ ] 抽出結果は画像生成向けのコーデ再現プロンプトとして表示できる
- [ ] アニメ風イラスト画像を入力した場合、画風ではなく服装構成を中心にした抽出結果になる
- [ ] 抽出処理に失敗した場合、失敗理由をセッションに記録できる

### ChatGPT比較評価記録

- [ ] ユーザーはChatGPTで得た抽出結果テキストを保存済みセッションに追加できる
- [ ] 本ツールの抽出結果とChatGPTの抽出結果を同一セッション内で確認できる
- [ ] 比較観点ごとに `同等`、`不足`、`過剰`、`要改善` のいずれかを記録できる
- [ ] 比較評価には自由記述メモを追加できる
- [ ] 比較評価は入力画像と抽出結果に紐づいて保存される

### 抽出セッションのローカル保存

- [ ] 抽出セッションは `data/sessions/` 配下に保存される
- [ ] 保存データには入力画像パス、入力種別、抽出結果、比較結果、作成日時、更新日時が含まれる
- [ ] 保存済みセッションの一覧を確認できる
- [ ] 保存済みセッションの詳細を確認できる

## スコープ外

以下はこの作業では実装しない:

- 固定人物プロンプトの登録・編集・削除
- ネガティブ抑制プロンプトの登録・編集・削除
- 固定人物プロンプト、コーデ抽出プロンプト、ネガティブ抑制プロンプトの最終合成
- ChatGPT貼り付け用の最終画像生成プロンプト出力
- 画像生成APIの呼び出し
- 生成済み画像の自動評価
- 背景、撮影、ポーズプロンプトの抽出
- SQLiteによる永続化
- GUI、TUI、ローカルWeb UI
